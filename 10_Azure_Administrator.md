# Microsoft Azure Administrator (AZ-104) Guide (2025-2026)

## Overview
Azure Administrator certification validates expertise in managing Azure resources, implementing and maintaining cloud infrastructure, and managing access and security.

**Certification Code:** AZ-104

---

## Duration & Study Timeline
- **Recommended Study Duration:** 2-3 months
- **Study Hours Required:** 100-150 hours
- **Hands-On Lab Time:** 50-80 hours
- **Prerequisites:** Azure Fundamentals (AZ-900) recommended

---

## Exam Details

### Exam Duration
- **Total Duration:** 120 minutes
- **Total Questions:** 40-60 questions
- **Passing Score:** 700/1000
- **Question Format:** Multiple choice, multiple select, hotspot, drag-and-drop

### Exam Fee (2025-2026)
- **Exam Fee:** $165 USD
- **Retake Fee:** $165 USD
- **Microsoft Learn (free):** Study modules
- **Lab Environment:** Azure free account ($200 credit)

## Complete Syllabus & Topics

### Domain 1: Manage Azure Identities & Governance (20-25%)

#### Azure AD & Access Management
- **Azure Active Directory (Entra ID)**
  - User and group management
  - User roles and administrative roles
  - Self-service password reset
  - Multi-factor authentication (MFA)
  - Conditional access policies
  - Identity protection
- **Authentication & Authorization**
  - Single sign-on (SSO)
  - OAuth and OpenID Connect
  - Service principals
  - Managed identities
  - RBAC (Role-Based Access Control)
  - Custom roles

#### Governance & Compliance
- **Azure Governance**
  - Subscriptions and management groups
  - Policies and policy effects
  - Policy scope and assignments
  - Exemptions
  - Monitoring and compliance
- **Cost Management**
  - Cost analysis
  - Budget alerts
  - Reserved instances
  - Spot VMs
  - Pricing calculator

### Domain 2: Implement & Manage Storage (20-25%)

#### Azure Storage Accounts
- **Storage Services**
  - Blob storage
  - File shares (SMB, NFS)
  - Queue storage
  - Table storage
  - Azure Data Lake Storage
- **Storage Configuration**
  - Access tiers (hot, cool, archive)
  - Replication strategies
  - Redundancy options (LRS, GRS, RAGRS, GZRS)
  - Storage security
  - Encryption at rest and in transit
- **Storage Management**
  - Lifecycle policies
  - Blob snapshots
  - Blob soft delete
  - Azure File Sync

#### Data Protection
- **Backup & Recovery**
  - Azure Backup
  - Recovery Services vault
  - Backup retention policies
  - Point-in-time recovery
  - Geo-redundancy
- **Azure Site Recovery**
  - Disaster recovery setup
  - Replication policies
  - Failover and failback
  - Recovery plans

### Domain 3: Deploy & Manage Azure Compute (20-25%)

#### Virtual Machines
- **VM Management**
  - Creating and configuring VMs
  - VM sizing and performance
  - Managed disks
  - Custom script extensions
  - Desired State Configuration (DSC)
  - VM networking
- **VM Availability**
  - Availability sets
  - Availability zones
  - Update domains
  - Fault domains
  - Virtual Machine Scale Sets (VMSS)

#### Containers & App Services
- **App Service**
  - App Service plans
  - Web apps
  - API apps
  - Function apps
  - Authentication and authorization
- **Container Services**
  - Azure Container Registry
  - Azure Container Instances
  - Azure Kubernetes Service (AKS)
  - Container troubleshooting

### Domain 4: Configure & Manage Virtual Networking (20-25%)

#### Virtual Networks
- **VNet Design**
  - Virtual networks and subnets
  - CIDR planning
  - IP addressing
  - Service endpoints
  - Private endpoints
  - Azure Private Link
- **Network Connectivity**
  - VNet peering
  - VNet-to-VNet connections
  - Site-to-site VPN
  - ExpressRoute
  - VPN Gateway
  - Application Gateway

#### Network Security
- **Security Components**
  - Network Security Groups (NSG)
  - Application Security Groups (ASG)
  - Azure Firewall
  - DDoS protection
  - Web Application Firewall (WAF)
  - DNS protection

#### DNS & Load Balancing
- **DNS Services**
  - Azure DNS
  - DNS zone configuration
  - DNS records
  - Health checks
- **Load Balancing**
  - Load Balancer
  - Application Gateway
  - Traffic Manager
  - Frontend/backend pool configuration

### Domain 5: Monitor & Maintain Azure Resources (10-15%)

#### Monitoring & Diagnostics
- **Azure Monitor**
  - Metrics and logs
  - Log Analytics
  - KQL (Kusto Query Language)
  - Alerts and action groups
  - Diagnostic settings
  - Application Insights
  - VM extensions for monitoring
- **Health & Performance**
  - Resource health
  - Service health
  - Performance monitoring
  - Baseline metrics
  - Anomaly detection

#### Maintenance & Updates
- **Update Management**
  - Patch management
  - Update schedules
  - Compliance monitoring
  - Maintenance windows
- **System Maintenance**
  - Backup verification
  - Disaster recovery drills
  - Configuration backup
  - Recovery procedures
