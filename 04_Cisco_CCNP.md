# Cisco CCNP Certification Guide (2025-2026)

## Overview
Cisco CCNP (Cisco Certified Network Professional) is a professional-level certification validating advanced networking knowledge including enterprise infrastructure, security, and automation. It focuses on implementing, operating, and troubleshooting Cisco networks at an enterprise scale.

**Certification Type:** Multi-exam concentration program

---

## Certification Paths Available (2025-2026)

### Enterprise Core & Concentrate Tracks
1. **Enterprise Infrastructure (ENCOR 350-401 + ENARSI 300-410)**
2. **Enterprise Security (ENCOR 350-401 + ENSECURITY 300-710)**
3. **Service Provider (ENCOR 350-401 + SPCOR 300-500 or SPRI 300-540)**
4. **Collaboration (ENCOR 350-401 + CLCOR 350-801)**
5. **Data Center (ENCOR 350-401 + DCCORE 350-601)**

---

## Duration & Study Timeline
- **Recommended Study Duration:** 6-12 months
- **Study Hours Required:** 400-500 hours per exam
- **Hands-On Lab Time:** 150-200 hours
- **Total Study Time:** 1-2 years for mastery

## Exam Details

### Core Exam: ENCOR (350-401)
- **Duration:** 120 minutes
- **Questions:** 60-70 items
- **Passing Score:** 825/1000
- **Fee:** $330 USD
- **Format:** Multiple choice, drag-and-drop, simulation, testlet

### Concentration Exams (varies by track)
- **Duration:** 90-120 minutes
- **Questions:** 50-65 items
- **Passing Score:** 750-825/1000
- **Fee:** $300-330 USD per exam
- **Format:** Similar to core exam

### Total CCNP Cost
- **Two Exam Path:** $630-660 USD
- **Retake Fee:** $330 USD per exam
- **Study Materials:** $50-200+ depending on resources

## Complete Syllabus & Topics

### ENCOR (Enterprise Core) 350-401 - 40% Weight

#### Architecture (15%)
- **Network Architecture**
  - Cisco Enterprise Architecture
  - Enterprise wired and wireless design
  - Data center design basics
  - Cloud connectivity design
  - Branch office connectivity
  - Security architecture
- **Model-Driven Telemetry**
  - Telemetry concept and benefits
  - Model-Driven Telemetry (MDT)
  - Push vs pull models
  - Telemetry agents and collectors
  - Data formats (gRPC, MQTT, REST)

#### Virtualization (15%)
- **Network Virtualization**
  - Virtual routing and forwarding (VRF)
  - Overlay networks
  - Network virtualization platforms
  - Container networking
  - Virtual network functions (VNF)
- **Infrastructure as Code**
  - Ansible for networking
  - Configuration management
  - Orchestration basics
  - API-driven infrastructure

#### Infrastructure (20%)
- **High Availability**
  - Redundancy concepts
  - First Hop Redundancy Protocol (FHRP)
  - HSRP, VRRP, GLBP
  - Stateful failover
  - BFD (Bidirectional Forwarding Detection)
- **Device Resilience**
  - Chassis redundancy
  - Module redundancy
  - Graceful restart (GR)
  - NSF/NDU (Non-Stop Forwarding/Upgrade)
  - Multicast protocols

#### Network Assurance (15%)
- **Network Monitoring**
  - Cisco DNA Center
  - Cisco Catalyst Center
  - NetFlow and Sflow
  - SNMP v3
  - Syslog and logging
  - Time synchronization (NTP)
- **Analytics**
  - Flow analysis
  - Network performance monitoring
  - Anomaly detection
  - Packet capture and analysis
  - Application visibility

#### Security (20%)
- **Advanced ACLs**
  - Complex ACL scenarios
  - IPv6 ACLs
  - Object groups
  - Reflexive ACLs
- **Device Hardening**
  - User authentication
  - Command authorization (RBAC)
  - Encrypted management access
  - SSH and secure protocols
  - Service security
- **Encryption**
  - IPSec site-to-site VPN
  - Crypto policies
  - PKI implementation
  - Digital certificates
- **Threat Defense**
  - IDS/IPS concepts
  - Anti-malware
  - Advanced threat protection

#### Services (15%)
- **QoS (Quality of Service)**
  - QoS mechanisms
  - Traffic policing and shaping
  - Queueing algorithms
  - CBWFQ, LLQ, WRED
  - QoS verification
- **Service Scalability**
  - Load balancing concepts
  - Server load balancing
  - Global load balancing
  - Connection management
  - Network address translation advanced

## Certification Validity & Renewal

### CCNP Validity
- **Certification Period:** 3 years
- **Renewal Methods:**
  - Pass new ENCOR exam OR
  - Pass any CCNP exam OR
  - Pass any CCIE exam OR
  - Maintain Cisco Learning Subscription (CLC)

### Maintenance Exam
- **Duration:** 1 hour
- **Format:** Exam covering all CCNP topics
- **Fee:** $330 USD
- **Alternative:** Take full CCNP path again

## Exam Tips & Strategy

1. **ENCOR First:** Master core concepts before specialization
2. **Realistic Labs:** Use Cisco Modeling Labs or GNS3
3. **Troubleshooting Focus:** Practice diagnostics extensively
4. **Time Management:** ~100 seconds per question
5. **Flag & Review:** Mark difficult questions for later
6. **Trust Training:** You've prepared for this
7. **Stay Current:** Follow Cisco updates and new technologies
