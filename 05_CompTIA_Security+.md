# CompTIA Security+ Certification Guide (2025-2026)

## Overview
CompTIA Security+ validates IT security skills covering security architecture, cryptography, identity and access management, risk management, and incident response. It's recognized by DoD 8570 standards.

**Certification Code:** SY0-701

---

## Duration & Study Timeline
- **Recommended Study Duration:** 3-4 months
- **Study Hours Required:** 150-200 hours
- **Exam Preparation Time:** 4-6 weeks

---

## Exam Details

### Exam Duration
- **Total Duration:** 90 minutes
- **Total Questions:** 79-86 questions
- **Question Format:** Multiple choice, performance-based questions (PBQ)

### Exam Structure
- **Passing Score:** 750/900
- **Score Range:** 100-900
- **Difficulty Level:** Intermediate Security

### Exam Fee (2025-2026)
- **Exam Fee:** $404 USD
- **Retake Fee:** $404 USD
- **Bundle with Network+/Pentest+:** Available
- **Testing:** Pearson VUE worldwide

## Complete Syllabus & Topics

### Domain 1: General Security Concepts (12%)

#### CIA Triad & Security Principles
- **Confidentiality, Integrity, Availability**
  - Data protection mechanisms
  - Access controls
  - Data validation
  - System availability measures
- **Defense in Depth**
  - Layered security approach
  - Multiple control types
  - Risk mitigation strategies
- **Zero Trust Model**
  - Continuous verification
  - Microsegmentation
  - Least privilege access
  - Never trust, always verify

#### Common Threats
- **Malware Types**
  - Viruses, worms, trojans
  - Ransomware and cryptolocker
  - Spyware and adware
  - Rootkits and bootkit
  - Logic bombs
- **Threat Actors**
  - Motivations (financial, political, ideological)
  - Skill levels (script kiddies to nation states)
  - Insider threats
  - External threats
- **Attack Methods**
  - Phishing and spear-phishing
  - Social engineering
  - Credential stuffing
  - Brute force attacks
  - Supply chain attacks

### Domain 2: Threat, Vulnerability & Risk Management (22%)

#### Risk Assessment
- **Risk Concepts**
  - Risk identification and analysis
  - Quantitative risk assessment
  - Qualitative risk assessment
  - Risk matrices and ratings
  - Asset valuation
  - Threat evaluation
  - Vulnerability scoring (CVSS)
- **Risk Response**
  - Risk mitigation
  - Risk acceptance
  - Risk avoidance
  - Risk transference (insurance)

#### Vulnerability Management
- **Vulnerability Assessment**
  - Scanning techniques
  - Patch management
  - Configuration review
  - Penetration testing
  - Vulnerability databases (NVD, CISA)
- **Threat Modeling**
  - Threat identification
  - Attack paths
  - Impact analysis
  - STRIDE methodology
  - Attack trees

### Domain 3: Security Architecture (15%)

#### Network Security Architecture
- **Network Segmentation**
  - DMZ (Demilitarized Zone)
  - VLANs
  - Subnetting
  - Microsegmentation
  - Zero Trust Network Architecture
- **Network Components**
  - Firewalls and next-gen firewalls
  - IDS/IPS systems
  - Proxies and reverse proxies
  - Load balancers
  - Network monitoring tools

#### Cloud & Virtualization Security
- **Cloud Security**
  - Shared responsibility model
  - Cloud service models (IaaS, PaaS, SaaS)
  - Cloud access security brokers (CASB)
  - Data residency and sovereignty
- **Virtualization**
  - Hypervisor security
  - Virtual machine isolation
  - Container security
  - Microservices security

#### Secure Design Principles
- **Secure by Design**
  - Privacy by design
  - Secure defaults
  - Fail securely
  - Least privilege
  - Separation of duties
  - Defense in depth

### Domain 4: Identity & Access Management (16%)

#### Authentication
- **Authentication Factors**
  - Something you know (passwords, PINs)
  - Something you have (tokens, smart cards)
  - Something you are (biometrics)
  - Somewhere you are (location)
  - Something you do (behavior)
- **Authentication Mechanisms**
  - Password policies
  - Multi-factor authentication (MFA)
  - Single sign-on (SSO)
  - RADIUS and TACACS+
  - OAuth and OpenID Connect
  - SAML
  - Passwordless authentication
  - Biometric systems

#### Authorization & Access Control
- **Access Control Models**
  - Role-Based Access Control (RBAC)
  - Attribute-Based Access Control (ABAC)
  - Access Control Lists (ACLs)
  - Discretionary Access Control (DAC)
  - Mandatory Access Control (MAC)
- **Privilege Management**
  - Least privilege principle
  - Privileged Account Management (PAM)
  - Sudo and elevation controls
  - Session management
  - Account provisioning/deprovisioning

### Domain 5: Cryptography & PKI (12%)

#### Cryptographic Concepts
- **Encryption Types**
  - Symmetric encryption (AES, 3DES)
  - Asymmetric encryption (RSA, ECC)
  - Hashing algorithms (SHA-256, MD5)
  - Cipher modes
  - Key management
- **Cryptographic Applications**
  - Data at rest encryption
  - Data in transit encryption
  - Digital signatures
  - Message authentication codes
  - Key derivation functions
  - Perfect forward secrecy

#### PKI & Certificates
- **Public Key Infrastructure**
  - Certificate Authority (CA)
  - Registration Authority (RA)
  - Certificate lifecycle
  - Key escrow
  - Trust models
- **Digital Certificates**
  - X.509 certificates
  - Certificate formats
  - Certificate validation
  - CRL and OCSP
  - Certificate pinning
- **Common Uses**
  - HTTPS/TLS
  - Email encryption (S/MIME)
  - Code signing
  - Blockchain security
